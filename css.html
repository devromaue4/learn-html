Inheritance

В CSS существует огромное количество свойств. Их можно разбить на следующие группы:

  • оформление текста;
  • работа с размерами и отступами;
  • позиционирование элементов;
  • работа с сеткой страницы;
  • декоративные: цвета, фон, тени;
  • другие.
